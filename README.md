# 电子发票智能管理系统 (Scan_FaPiao)

这是一个基于 Vite + React 构建的高效、智能的电子发票扫描与管理系统。它旨在简化财务报销和发票查验流程，通过自动化的方式处理多种格式的发票文件。

## 🚀 核心功能

- **全格式支持**：支持 PDF、OFD 以及常见的图片格式 (JPG, PNG, BMP 等) 电子发票上传。
- **智能查验**：对接第三方查验接口，自动识别发票真伪及明细信息。
- **全局拖拽上传**：支持在页面任意位置拖入文件进行上传，极大提升交互体验。
- **数据持久化**：扫描记录、发票明细及商品明细自动同步至 Jetop CMS 数据区块。
- **统计面板**：实时展示累计识别、查验成功及失败的数量统计。
- **高级搜索**：支持按发票号码、代码、销方/购方名称、日期及金额等多维度进行前端实时过滤。
- **批量操作**：支持发票记录的批量勾选、批量删除以及数据导出功能。
- **沉浸式预览**：提供发票原件的在线预览，支持图片缩放、全屏查看及源文件下载。

## 🛠 技术栈

- **前端框架**：React 18
- **构建工具**：Vite
- **图标库**：React Icons
- **文件指纹**：Spark-MD5 (用于文件去重检测)
- **样式处理**：原生 CSS (自定义设计系统)

## 📦 安装与运行

1. 克隆仓库：
   ```bash
   git clone https://github.com/supernova-ovo/Scan_FaPiao.git
   ```
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm run dev
   ```
4. 构建生产版本：
   ```bash
   npm run build
   ```
